📋 Términos de Servicio

1. Servicios Ofrecidos

DarDev proporciona servicios de desarrollo de software multiplataforma, incluyendo aplicaciones móviles (Android/iOS), desarrollo web, software de escritorio, integración de APIs y manejo de bases de datos.

2. Condiciones Generales

  • Los proyectos se desarrollan según especificaciones técnicas acordadas por escrito
  • Los plazos de entrega se establecen según la complejidad del proyecto
  • Se requiere el 50% de anticipo para iniciar cualquier proyecto
  • Los cambios en el alcance del proyecto pueden afectar costos y tiempos

3. Derechos de Propiedad

  • El cliente obtiene derechos completos sobre el código desarrollado una vez completado el pago
  • DarDev mantiene derechos sobre metodologías y frameworks propios
  • Se proporcionará documentación técnica y código fuente al finalizar

4. Soporte y Mantenimiento

  • Se incluyen 30 días de soporte gratuito post-entrega
  • Corrección de bugs identificados durante el período de garantía
  • Soporte extendido disponible bajo acuerdo separado
